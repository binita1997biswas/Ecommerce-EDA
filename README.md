# Ecommerce Exploratory Data Analysis (EDA)
## Project Overview
This project performs Exploratory Data Analysis (EDA) on an Ecommerce dataset to identify customer purchasing patterns, product performance, and transaction insights.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Analysis
- Data Cleaning
- Missing Value Analysis
- Duplicate Removal
- Datatype Conversion
- Customer Analysis
- Product Analysis
- Transaction Analysis
- Correlation Analysis
- Visualizations

## Dataset
Online Retail Dataset
